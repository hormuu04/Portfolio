# อธิบายโปรเจค Phulaport

เว็บไซต์ Portfolio นี้ผมสร้างขึ้นมาเพื่อแนะนำตัวและแสดงทักษะการเขียนโปรแกรมของผม รวมถึงรวบรวมโปรเจคต่างๆ ที่ผมเคยทำไว้ในเว็บไซต์เดียว

เว็บไซต์นี้พัฒนาโดยใช้ **JavaScript** และใช้ **React Framework** ร่วมกับ **Tailwind CSS Framework** และยังมีไลบรารีอื่นๆ ที่ช่วยเสริมการทำงานอีกด้วย

## เหตุผลที่เลือกใช้เทคโนโลยีเหล่านี้

### 1. React
React คือ "HTML ที่เขียนด้วย JavaScript" ซึ่ง React มีความสามารถในการแยกส่วนต่างๆ ของเว็บออกเป็น **Components** เล็กๆ ทำให้เราสามารถพัฒนาแต่ละส่วนแยกกันได้ เช่น ส่วน "Navbar" และส่วน "Project" ไม่ปะปนกัน ทำให้โค้ดอ่านง่ายขึ้น และเมื่อพัฒนาแต่ละส่วนเสร็จแล้วก็จะนำมารวมกันในแอปพลิเคชันหลัก

### 2. Tailwind CSS
Tailwind CSS เป็น **CSS Framework** ที่ช่วยให้เราเขียน CSS ได้อย่างง่ายและกระชับ โดย Tailwind จะทำให้เราไม่ต้องสร้างไฟล์ CSS แยก แต่สามารถเขียนสไตล์ได้เลยใน `Class` attribute ของ HTML Element ในบรรทัดเดียวกัน ซึ่งช่วยประหยัดเวลาได้มาก

### 3. Framer Motion
ผมได้ลองใช้ไลบรารีใหม่ชื่อ **Framer Motion** ซึ่งเป็นไลบรารีที่ช่วยในการออกแบบและควบคุม **Animation** ขององค์ประกอบต่างๆ ได้อย่างง่ายดาย ผมคิดว่ามันช่วยเพิ่มความสวยงามให้กับเว็บไซต์สำหรับผู้เยี่ยมชมได้เป็นอย่างดี

### 4. AOS (Animate On Scroll)
ผมใช้ไลบรารี **AOS** เพื่อเพิ่มเอฟเฟกต์การแสดงผลเมื่อผู้ใช้เลื่อนหน้าจอ ทำให้เว็บไซต์ดูน่าสนใจและมีชีวิตชีวามากขึ้น โดยแต่ละส่วนจะมีการแสดงผลแบบ fade-in, zoom-in ตามที่กำหนดไว้

### 5. Vite
ใช้ **Vite** เป็น Build Tool หลัก เพราะ Vite มีความเร็วในการพัฒนาและ build โปรเจคสูงมาก ช่วยให้การพัฒนาเว็บไซต์รวดเร็วและมีประสิทธิภาพ
